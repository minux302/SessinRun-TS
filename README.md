# SessionRun-TS

## Requirements

- Node.js 8.11.1

## Setup

1. put `songs` and `web_model` in `dist` (This step will be removed in the future.)
2. install dependencies

```sh
$ yarn
```

## How to start

```sh
$ yarn start
```

## Run formatter

```sh
$ yarn fmt
```

## Build

```sh
$ yarn build
```
